\# SEO Audit Tool



\## What This Tool Does

A local SEO audit and website diagnostic tool built for Elitez.



Scans websites for:

\- Technical SEO issues

\- Missing H1 / Meta / Alt Text

\- Broken links

\- Robots / Sitemap / Crawlability

\- Conversion / Messaging issues



\---



\## How To Run



1\. Open terminal

2\. Go into project folder:



cd "C:\\Users\\Keok\\Desktop\\seo-audit-tool"



3\. Start server:



node server.js



4\. Open browser:



http://localhost:3000
