Set Up Google Search Console

Pull real search data — queries, rankings, clicks, and CTR — directly from Google. No third-party tools. No API limits beyond Google's own.

  1. Go to Google Cloud Console → Credentials Create a new project (or use an existing one)
  2. Enable the Google Search Console API APIs & Services → Library → search "Search Console API" → Enable
  3. Create OAuth 2.0 credentials Credentials → Create Credentials → OAuth client ID → Web application
  4. Add the redirect URI http://localhost:3000/auth/google/callback
  5. Add credentials to your .env file, then restart the server GSC_CLIENT_ID=your_client_id
    GSC_CLIENT_SECRET=your_client_secret
